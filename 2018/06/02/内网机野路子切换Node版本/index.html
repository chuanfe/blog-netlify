内网机野路子切换Node版本

Jun 2, 2018

公司电脑连不上外网，用不了n模块这样的工具切换node版本，只能暴力切换。下面是暴力切换的整个过程，主要是三步：下载zip包，本地新建一个文件夹放各个版本，更改path

node官网下载对应版本

https://nodejs.org/zh-cn/download/releases/
直接下载zip包，不要下载安装程序

本地新建一个文件夹管理各版本

解压刚才下载的版本，可以重命名文件夹方便后续切换版本

各node版本

更改path环境变量

新建一个变量名NODE_HOME，path加的时候注意前后％号和;号 ;%NODE_HOME%;
